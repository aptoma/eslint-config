# @aptoma/eslint-config

Shareable ESLint config following Aptoma Standard.

## Installation

```
$ npm install --save-dev @aptoma/eslint-config
```  

In your ESLint configuration add:

```
{
  "extends": "@aptoma/eslint-config"
}
```
